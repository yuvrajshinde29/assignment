					-The method is case-sensitive.
					-Returns a boolean: true
				Ex-	
					const str = "Hello world, welcome to the universe.";

					console.log(str.startsWith("Hello"));   // true
					console.log(str.startsWith("world"));   // false

					console.log(str.startsWith("hello"));   // false (case-sensitive)

			push():
				add  one or more element to the end of array.
				it modifies the original array and return new length of array.
				
				array.push(element1, element2, ..., elementN)

					let fruits = ["Banana", "Orange", "Apple"];
					
					let newLength = fruits.push("Mango");
					console.log(fruits);      // ["Banana", "Orange", "Apple", "Mango"]
					console.log(newLength);   // 4
					
					fruits.push("Pineapple", "Grapes");
					console.log(fruits);      // ["Banana", "Orange", "Apple", "Mango", "Pineapple", "Grapes"]
					console.log(fruits.length); // 6
					
				
			----------------------------------------------------------------------------------------------
			pop():
				It removes and returns the last element of the array.
				It changes the original array.
				If the array is empty, it returns undefined.
				It takes no arguments.
				Commonly used to implement stack behavior (LIFO - Last In, First Out)
				
					let fruits = ["Banana", "Orange", "Apple", "Mango"];
					
					let lastFruit = fruits.pop();
					
					console.log(lastFruit);  // Output: Mango
					console.log(fruits);     // Output: ["Banana", "Orange", "Apple"]
					
			----------------------------------------------------------------------------------------------
			shift():
			
				Removes and returns the first element of the array.
				Modifies the original array and reduces its length by one.			
				If called on an empty array, it returns undefined.			
				Shifts all subsequent elements to lower indexes (index 1 becomes index 0, index 2 becomes 1, etc.).
				
					const fruits = ["Banana", "Orange", "Apple", "Mango"];
					const firstFruit = fruits.shift();
					
					console.log(firstFruit);  // Output: "Banana"
					console.log(fruits);      // Output: ["Orange", "Apple", "Mango"]
					
			----------------------------------------------------------------------------------------------
			unshift():
				The unshift() method in JavaScript adds one or more elements to the beginning of an array and returns the new length of the array
				
					let fruits = ["Apple", "Banana"];
					let newLength = fruits.unshift("Strawberry", "Orange");
					
					console.log(fruits);    // ["Strawberry", "Orange", "Apple", "Banana"]
					console.log(newLength); // 4

			splice():
				method that can add, remove, or replace elements within an array, modifying the original array directly
				
					array.splice(start, deleteCount,item1, item2, ...itemN);
					
					Removing Elements:
						let arr = ["Apple", "Banana", "Cherry", "Date"];
						let removed = arr.splice(1, 2); // removes 2 elements starting from index 1
						
						console.log(arr);     // ["Apple", "Date"]
						console.log(removed); // ["Banana", "Cherry"]
						
					-------------------------------------------
					Adding Elements:
						let arr = ["Apple", "Banana", "Date"];
						arr.splice(2, 0, "Cherry", "Mango"); // add without removing
						
						console.log(arr); // ["Apple", "Banana", "Cherry", "Mango", "Date"]
					-------------------------------------------
					Replacing Elemenys:
						let arr = ["Apple", "Banana", "Cherry", "Date"];
						arr.splice(1, 2, "Kiwi", "Peach"); // remove 2 elements at index 1, add 2 new elements
						
						console.log(arr); // ["Apple", "Kiwi", "Peach", "Date"]
					-------------------------------------------
					-Negative start counts from the end (e.g., -1 is last element).

			slice():
				The slice() method in JavaScript returns a shallow copy of a portion of an array into a new array, selected from a start index up to but not including an end index. It does not modify the original array.
				
					array.slice(start, end)

				start (optional): Zero-based index at which to begin extraction. Defaults to 0. Can be negative to count from the end.

				end (optional): Zero-based index before which to end extraction (not included). Defaults to array length. Can be negative.
				
				Ex-
				const fruits = ['apple', 'banana', 'cherry', 'date', 'elderberry'];
				
				console.log(fruits.slice());		//[ 'apple', 'banana', 'cherry', 'date', 'elderberry' ]
				console.log(fruits.slice(1, 3)); 	//['banana', 'cherry']
				console.log(fruits.slice(2));		//['cherry', 'date', 'elderberry']
				console.log(fruits.slice(-3, -1));	//['cherry', 'date']

	-Function :
		block of code use to perform a task or repeated task
		Functions are first class objects in javaScript
		
			call():
					functionName.call(thisArg, arg1, arg2, ..., argN)

				-use to explicitly set the value of this inside a function.
				-call the function immediately
				-arguments are passed individually, seperated by commas.
				
				Ex-
					let person1 ={
						name : "raj",
						greet : function (greetings){
							console.log(`${greetings} ${this.name}`);
						}
					}
					
					let person2 = {
						name :"virat"
					}
					// Call greet of person1 but with person2 as `this`
					person1.greet.call(person2, "good morning");	//good morning virat
					
					------
				-It enables you to call a function that belongs to one object with another object as its this value.
				-Useful for setting the context (this) explicitly when calling a function.
				-Allows passing arguments one by one after the this argument.
			----------------------------------------------------------------------------------------------
			apply():
				
				The apply() method in JavaScript calls a function with a specified this value and arguments provided as an array (or an array-like object).
				
					func.apply(thisArg, argsArray)

					Ex-
						------
						const person = {
						firstName: "Taylor",
						lastName: "Jackson"
						};
						
						function greet(wish, message) {
						return `${this.firstName}, ${wish}. ${message}`;
						}
						
						const result = greet.apply(person, ["Good morning", "How are you?"]);
						console.log(result); 
						// Output: Taylor, Good morning. How are you?
						----
			
				-Calls the function immediately.
				-Takes the this value as the first argument.
				-Arguments are passed as a single array.
			----------------------------------------------------------------------------------------------
			bind():				
				Returns a new function with bound this and optionally preset initial arguments.
				Does not call the function immediately.
				Arguments can be provided when calling this new function.
				
				func.bind(thisArg[, arg1[, arg2[, ...]]])
	
				Ex-
					let person1 ={
						name:"rohit",
						greet: function(greetings, message){
							console.log(`${this.name} : ${greetings} . ${message}`);
						}
					}
					
					let person2={
						name:"virat"
					}
					
					let newFun =person1.greet.bind(person2,"good morning", "how are you?");
					newFun();
					---or---
