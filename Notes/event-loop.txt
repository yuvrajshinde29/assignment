Execution Context:
---------
https://www.freecodecamp.org/news/execution-context-how-javascript-works-behind-the-scenes/
----------
execution context is a conceptual environment where JavaScript code is evaluated and executed.
types of execution context:
	Global Execution Context:
		 The base execution context for any JavaScript code that is not inside a function. There is only one GEC per JavaScript file.
	FUnction execution context:
		Created whenever a function is invoked. Each function call gets its own FEC
